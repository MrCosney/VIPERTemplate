# VIPERTemplate
_____ 
## How To Install

Clone the repository

Navigate to Xcode Templates folder: ~/Library/Developer/Xcode/Templates/. If Templates folder doesn't exist, create it

Copy and paste the VIPER Module.xctemplate in Templates folder
